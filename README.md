# Instruction-Simplifier-for-Neurodivergent-Students
04/15/2026 - 05/15/2026

An accessibility-first web app that transforms dense school instructions (syllabi, assignment prompts, rubric text, emails) into clear, manageable, neurodivergent-friendly formats: plain-language summaries, step-by-step checklists, “do this first” starters, and time/break plans.
